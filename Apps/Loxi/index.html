Features

macOS-only, built for Xcode localization files (.xcstrings).

Built for macOS & Xcode
Designed specifically for Apple developers.
Native .xcstrings support
Works directly with Xcode localization files.
Batch translation
Translate hundreds of strings in one action.
Safe placeholder handling
Preserves %@, %d, %1$s, and other format specifiers.
Instant missing detection
Spot untranslated strings at a glance.
Add/remove languages & keys
Full control over your localization structure.
Keyboard shortcuts (⌘ + T)
Translate faster without leaving the keyboard.
Side-by-side comparison
Review base strings and translations in one view.
Context-aware translations
Uses keys and comments to improve accuracy.
Native macOS experience
